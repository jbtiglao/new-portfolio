About Me

Hello, I'm Jane.

I am a web developer with a technical writing and legal background. I moved to California in 2008 and since then have gone back to law school, in the hopes of one day taking the Bar, while raising my two amazing sons. However, the events of the past years have made me look at transitioning into a career in tech and the possibility of doing remote work so I can spend more time with my family. As a former technical writer, I have always enjoyed working with developers. This time, it is an unexpected but exciting change to be working on the development side.

As a web developer and an Autism mom, I am particularly interested in accessibility. With my legal and special education background, one of my goals is to build accessible legal and special education/needs applications.

When I'm not coding, you can find me tinkering in my garden, mixing paints and creating art with my sons, playing with makeup, or curled up in a corner with a cup of coffee and a good book.

Projects

Project 1: Food For Thought

This project is an intercative web application that allows users to search for recipes with a single word or name of ingredient. As a member of the team, I worked on the application's wireframe using Google Slides and mock-up using Figma, as well as the HTML and CSS files. I was also responsible for integrating Bulma into the HTML and CSS framework, and the application's responsiveness when viewed in different devices. Last, I wrote the application's README, User Story, and Acceptance Criteria, and prepared screenshots of its functionalities for documenation.

Homework 1: Password Generator

This project is a random password generator that showcases the primary skills our class has discussed on Javascript (Unit 3). The project consists of a starter code that we were tasked to modify to create an application that enables employees to gerenate random passwords based on a criteria they have selected. The app runs in a browser and features a dynamically updated HTML and CSS, powered by Javascript.

Homework 2: Workday Scheduler

For the unit on Third-Party APIs, this project is a simple calendar application or workday scheduler that allows a user to save events for each hour of the day by modifying starter code. The application runs in the browser and features dynamically updated HTML and CSS, powered by jQuery. The Moment.js library is used to work with date and time.
